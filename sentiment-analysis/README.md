# NLP Sentiment Analysis Engine

Multilingual sentiment analysis pipeline for processing customer feedback and social media posts.

## 🚀 Key Achievement
- Processed **1M+ documents daily** with **92% classification accuracy**.  

## 🛠️ Tech Stack
- BERT / Transformers  
- SpaCy NLP  
- FastAPI for serving  
- MongoDB  
- Docker
